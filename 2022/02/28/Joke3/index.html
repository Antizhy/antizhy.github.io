zhy语解释

信手粘来

发音: xìn shǒu zhān lái

含义: 表示信非常粘手，粘到手上拿不下来

阀值

发音: fá zhí

含义: 阀门的价值

没听清楚 / 太粗了 / 太泛了

发音: méi tīng qīng chu / tài cū le / tài fàn le

含义: 你做的不行

你看了几篇论文

发音: nǐ kàn le jǐ piān lùn wén
含义: 不许否定我，我才是对的

后果自负！ / 延毕!

发音: hòu guǒ zì fù / yán bì
含义: 表示气炸了，要用延毕要挟你

收到请回复！！！

发音: shōu dào qǐng huí fù
含义: 不能无视我，要惯着我

抓手

发音: zhuā shǒu
含义: 我只要抓手不要脸

含义: 表示我非常非常生气
